<h1>Resume Optimizer</h1>
<h2>Summary</h2>
<p>In the modern competitive world job applications are a task. This project helps to optimize your resume based on the Job description in question.
It gives you :
<li>ATS Score</li>
<li>Strengths</li>
<li>Areas for Improvement </li>
<li>Top Recommendations</li></p>
<h3>Libraries Used</h3>
<ul>
  <li><b>re</b> : Regular Expressions</li>
  <li><b>nltk</b> : Natural Language Processing</li>
  <li><b>gradio</b> : Interface </li>
  <li><b>PyPDf2</b> : reading PDFs</li>
  <li><b>docx</b> : reading documents</li>
</ul>
